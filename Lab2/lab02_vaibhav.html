Adding a height-dependent g slightly increases the fall time compared to using constant gravity. This is because as the object falls deeper, its distance from Earth’s center decreases, which reduces the local gravitational acceleration according to

g(r) = g0 * (g/R0)

This means the test mass experiences slightly weaker acceleration the further it falls, so it takes longer to reach the bottom. THat is why the fall time increased from 28.557 s (constant g) to 28.558 s (variable g), which is small yet a measurable difference.

Including drag greatly increases fall time to 83.542 s, nearly three times longer. This occurs because drag limits velocity to the terminal speed (50 m/s), slowing the mass's descent. Once the mass reaches that speed, it falls at a nearly constant rate, which drastically slows its descent compared to the drag-free case.

shaft_halfwidth = 2.5  

v_terminal = 50.0
alpha = g0 / (v_terminal**2)  

def coriolis_no_drag(t, Y):
    x, vx, y, vy = Y
    ax = 2 * Omega * vy
    ay = g0 - 2 * Omega * vx
    return [vx, ax, vy, ay]

def coriolis_with_drag(t, Y):
    x, vx, y, vy = Y
    ax_c = 2 * Omega * vy
    ay_c = g0 - 2 * Omega * vx
    speed = np.sqrt(vx**2 + vy**2)
    if speed < 1e-12:
        ax_d, ay_d = 0.0, 0.0
    else:
        ax_d = -alpha * speed * vx
        ay_d = -alpha * speed * vy
    return [vx, ax_c + ax_d, vy, ay_c + ay_d]

    return Y[2] - H

event_bottom.direction = 1

def orbital_period(mass, radius):
    return 2 * np.pi * radius / np.sqrt(ge * mass / radius)

def tunnel_ode_uniform(t, state):
    y, vy = state
    r = R_EARTH + y  
    grav_local = G0 * (r / R_EARTH)
    dydt = vy
    dvdt = -grav_local
    return [dydt, dvdt]

def event_center(t, state):
    return state[0] + R_EARTH
event_center.terminal = True
event_center.direction = 0  

def event_opposite_side(t, state):
    return state[0] + 2*R_EARTH
event_opposite_side.terminal = True
event_opposite_side.direction = 0

def run_part4_infinite_tunnel():
    t0, tmax = 0, 3000
    t_span = (t0, tmax)

    y_init = 0.0
    vy_init = 0.0
    init_state = [y_init, vy_init]

    sol = solve_ivp(
        tunnel_ode_uniform,
        t_span,
        init_state,
        max_step=0.5,
        dense_output=True,
        events=[event_center, event_opposite_side],
        rtol=1e-9
    )

    if len(sol.t_events[0]) > 0:
        t_center = sol.t_events[0][0]
        y_center, v_center = sol.sol(t_center)
        print(f"Reached Earth's center at t = {t_center:.3f} s, speed = {v_center:.3f} m/s")

def densityNorm(r, n):
    return (1 - (r**2 / Rearth**2))**n if r<=Rearth else 0.0

def massUnscaled(n):
    f = lambda x: 4*np.pi*x**2*(1 - (x**2/Rearth**2))**n

def dens(r,n0,exp0):

The time (T) for an object to cross a uniformly dense planet follows simple harmonic motion. That means the crossing time is inversely proportional to the square root of the planet's density (rho):

T ∝ 1 / √(ρ)

  • Higher density → Stronger gravitational pull → Shorter crossing time
  • Lower density → Weaker gravitational pull → Longer crossing time
